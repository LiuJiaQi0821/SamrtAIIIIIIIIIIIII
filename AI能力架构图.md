# AI能力架构图

## 系统概览

"小职引职业规划助手"是一个基于AI的职业规划系统，通过简历分析、智能对话、人岗匹配等核心能力，为用户提供个性化的职业发展建议。

## 核心技术特性

### 1. 流式输出 (SSE)
- AI对话采用 Server-Sent Events 协议
- 打字机式实时渲染，提升用户体验
- 减少等待感知

### 2. 容错机制
- 岗位卡片渲染 try-catch 包裹
- 渲染失败显示备用卡片
- 确保界面不崩溃

### 3. 逐步放宽策略
- 6种匹配策略，从严格到宽松
- 智能降级，确保有结果
- 用户友好的提示语

### 4. 数据库优化
- REST API 直接调用，突破 ORM 限制
- 批量分页查询，获取全量数据
- 多索引优化，提升查询效率
