# 数据库结构设计

## 数据库类型
- **PostgreSQL** (通过 Supabase 托管)
- **ORM**: Drizzle ORM

---

## 数据表概览

| 表名 | 说明 | 主要用途 | 记录数 |
|------|------|---------|--------|
| `health_check` | 健康检查表 | 系统监控用 | - |
| `jobs` | 岗位数据表 | 存储招聘岗位信息 | ~9958条 |
| `job_profiles` | 岗位画像表 | 存储岗位能力要求和画像 | - |
| `career_paths` | 岗位晋升路径表 | 存储岗位之间的晋升关系 | - |
| `job_transfer_graph` | 换岗路径图谱表 | 存储岗位之间的转岗关系 | - |

## 详细表结构

### 1. health_check (健康检查表)
**系统表，禁止修改**

| 字段名 | PostgreSQL类型 | Drizzle类型 | 说明 | 示例值 |
|--------|--------------|------------|------|--------|
| `id` | SERIAL | serial | 主键 | 1, 2, 3... |
| `updated_at` | TIMESTAMPTZ | timestamp | 更新时间(带时区) | 2024-01-15T10:30:00+08:00 |

---

### 2. jobs (岗位数据表)
**存储招聘岗位信息 - 核心业务表**

| 字段名 | PostgreSQL类型 | Drizzle类型 | 约束 | 说明 | 索引 | 示例值 |
|--------|--------------|------------|------|------|------|--------|
| `id` | SERIAL | serial | PRIMARY KEY | 主键ID | - | 1, 2, 3... |
| `job_title` | VARCHAR(255) | varchar(255) | NOT NULL | 岗位名称 | ✅ | "前端开发工程师", "产品经理" |
| `address` | VARCHAR(255) | varchar(255) | NOT NULL | 工作地点/地址 | ✅ | "杭州市余杭区", "北京市朝阳区" |
| `salary_range` | VARCHAR(100) | varchar(100) | - | 薪资范围 | - | "15K-25K·14薪", "20-35K" |
| `company_name` | VARCHAR(255) | varchar(255) | NOT NULL | 公司名称 | ✅ | "阿里巴巴", "字节跳动" |
| `industry` | VARCHAR(255) | varchar(255) | - | 所属行业 | ✅ | "互联网", "金融", "教育" |
| `company_size` | VARCHAR(100) | varchar(100) | - | 公司规模 | - | "1000-9999人", "500-1000人" |
| `company_type` | VARCHAR(100) | varchar(100) | - | 公司类型 | - | "民营", "合资", "国企" |
| `job_code` | VARCHAR(100) | varchar(100) | - | 岗位代码 | ✅ | "DEV001", "PM002" |
| `job_description` | TEXT | text | - | 岗位描述 | - | "负责前端开发，使用React..." |
| `update_date` | VARCHAR(50) | varchar(50) | - | 更新日期 | - | "2024-01-15" |
| `company_description` | TEXT | text | - | 公司描述 | - | "阿里巴巴是中国领先的..." |
| `source_url` | VARCHAR(500) | varchar(500) | - | 来源链接 | - | "https://example.com/job/123" |
| `created_at` | TIMESTAMPTZ | timestamp | NOT NULL, DEFAULT NOW | 创建时间 | - | 2024-01-15T10:30:00+08:00 |

**示例数据**:
```json
{
  "id": 1,

  "salary_range": "15K-25K·14薪",
  "company_name": "阿里巴巴",
  "industry": "互联网",
  "company_size": "10000人以上",
  "company_type": "民营",
  "job_code": "FE001",
  "job_description": "负责淘宝前端开发，使用React技术栈...",
  "update_date": "2024-01-15",
  "company_description": "阿里巴巴集团控股有限公司...",
  "source_url": "https://example.com/job/1",
  "created_at": "2024-01-15T10:00:00+08:00"
}
```

### 3. job_profiles (岗位画像表)
**存储岗位能力要求和画像 - 岗位能力标准**

| 字段名 | PostgreSQL类型 | Drizzle类型 | 约束 | 说明 | 索引 | 示例值 |
|--------|--------------|------------|------|------|------|--------|
| `id` | SERIAL | serial | PRIMARY KEY | 主键ID | - | 1, 2, 3... |
| `profile_name` | VARCHAR(100) | varchar(100) | NOT NULL | 画像名称 | ✅ | "前端开发工程师", "产品经理" |
| `category` | VARCHAR(50) | varchar(50) | NOT NULL | 分类 | ✅ | "技术", "产品", "设计" |
| `description` | TEXT | text | - | 描述 | - | "负责Web前端开发工作..." |
| `professional_skills` | TEXT | text | - | 专业技能要求 | - | "React, Vue, TypeScript" |
| `certificate_requirements` | TEXT | text | - | 证书要求 | - | "计算机二级, PMP" |
| `innovation_ability` | INTEGER | integer | DEFAULT 3 | 创新能力(1-5分) | - | 3, 4, 5 |
| `learning_ability` | INTEGER | integer | DEFAULT 3 | 学习能力(1-5分) | - | 3, 4, 5 |
| `pressure_resistance` | INTEGER | integer | DEFAULT 3 | 抗压能力(1-5分) | - | 3, 4, 5 |
| `communication_ability` | INTEGER | integer | DEFAULT 3 | 沟通能力(1-5分) | - | 3, 4, 5 |
| `internship_requirement` | TEXT | text | - | 实习经验要求 | - | "1年以上前端开发经验" |
| `education_requirement` | VARCHAR(100) | varchar(100) | - | 学历要求 | - | "本科及以上" |
| `work_experience_requirement` | VARCHAR(100) | varchar(100) | - | 工作经验要求 | - | "3年以上" |
| `salary_range` | VARCHAR(100) | varchar(100) | - | 薪资范围 | - | "15K-30K" |
| `career_direction` | TEXT | text | - | 就业方向 | - | "前端架构师, 技术总监" |
| `core_qualifications` | TEXT | text | - | 核心素质要求 | - | "责任心强, 团队协作" |
| `tools_requirement` | TEXT | text | - | 工具/软件要求 | - | "VS Code, Git, Figma" |
| `created_at` | TIMESTAMPTZ | timestamp | NOT NULL, DEFAULT NOW | 创建时间 | - | 2024-01-15T10:30:00+08:00 |

### 4. career_paths (岗位晋升路径表)
**存储岗位之间的晋升关系 - 职业发展规划**

| 字段名 | PostgreSQL类型 | Drizzle类型 | 约束 | 说明 | 索引 | 示例值 |
|--------|--------------|------------|------|------|------|--------|
| `id` | SERIAL | serial | PRIMARY KEY | 主键ID | - | 1, 2, 3... |
| `from_job_id` | INTEGER | integer | NOT NULL, FK | 当前岗位ID (关联 job_profiles) | ✅ | 1, 2, 3... |
| `to_job_id` | INTEGER | integer | NOT NULL, FK | 目标岗位ID (关联 job_profiles) | ✅ | 5, 6, 7... |
| `path_type` | VARCHAR(20) | varchar(20) | NOT NULL | 路径类型 | - | "vertical"(垂直晋升), "horizontal"(横向转岗) |
| `difficulty` | VARCHAR(20) | varchar(20) | DEFAULT 'medium' | 难度 | - | "easy", "medium", "hard" |
| `years_required` | INTEGER | integer | DEFAULT 3 | 所需周期(年) | - | 1, 2, 3, 5 |
| `additional_skills` | TEXT | text | - | 所需补充技能 | - | "需要学习Node.js, 架构设计" |
| `additional_certificates` | TEXT | text | - | 所需补充证书 | - | "PMP项目管理认证" |
| `promotion_conditions` | TEXT | text | - | 晋升条件说明 | - | "完成3个大型项目, 团队绩效前20%" |
| `salary_changes` | TEXT | text | - | 薪资变化说明 | - | "预计涨幅30%-50%" |
| `path_description` | TEXT | text | - | 路径描述 | - | "从前端开发到高级前端的典型路径" |
| `is_recommended` | BOOLEAN | boolean | DEFAULT false | 是否推荐 | - | true, false |
| `created_at` | TIMESTAMPTZ | timestamp | NOT NULL, DEFAULT NOW | 创建时间 | - | 2024-01-15T10:30:00+08:00 |

**示例数据**:
```json
{
  "id": 1,
  "from_job_id": 1,
  "to_job_id": 5,
  "path_type": "vertical",
  "difficulty": "medium",
  "years_required": 3,
  "additional_skills": "需要学习Node.js后端开发, 系统设计",
  "additional_certificates": "推荐考取系统架构设计师证书",
  "promotion_conditions": "1. 担任2年以上前端开发\n2. 独立负责过3个以上大型项目\n3. 团队绩效考核优秀",
  "salary_changes": "预计薪资涨幅30%-50%, 从15K-25K涨到20K-35K",
  "path_description": "前端开发工程师 → 高级前端开发工程师的标准晋升路径",
  "is_recommended": true,
  "created_at": "2024-01-15T10:00:00+08:00"
}
```

### 5. job_transfer_graph (换岗路径图谱表)
**存储岗位之间的转岗关系 - 职业转换图谱**

| 字段名 | PostgreSQL类型 | Drizzle类型 | 约束 | 说明 | 索引 | 示例值 |
|--------|--------------|------------|------|------|------|--------|
| `id` | SERIAL | serial | PRIMARY KEY | 主键ID | - | 1, 2, 3... |
| `from_job_id` | INTEGER | integer | NOT NULL, FK | 起始岗位ID (关联 job_profiles) | ✅ | 1, 2, 3... |
| `to_job_id` | INTEGER | integer | NOT NULL, FK | 目标岗位ID (关联 job_profiles) | ✅ | 10, 11, 12... |
| `path_type` | VARCHAR(20) | varchar(20) | NOT NULL | 路径类型 | - | "related", "unrelated" |
| `intermediate_job_id` | INTEGER | integer | FK | 中间岗位ID (关联 job_profiles) | - | 5, 6, 7... |
| `transfer_difficulty` | VARCHAR(20) | varchar(20) | DEFAULT 'medium' | 转岗难度 | - | "easy", "medium", "hard" |
| `steps_required` | INTEGER | integer | DEFAULT 1 | 所需步骤数 | - | 1, 2, 3 |
| `additional_skills` | TEXT | text | - | 所需补充技能 | - | "需要学习产品思维, 数据分析" |
| `additional_certificates` | text | text | - | 所需补充证书 | - | "推荐PMP产品经理认证" |
| `transfer_conditions` | TEXT | text | - | 转岗条件说明 | - | "需要有产品相关项目经验" |
| `salary_impact` | TEXT | text | - | 薪资影响 | - | "初期可能降薪10%-20%, 长期看涨" |
| `transfer_tips` | TEXT | text | - | 转岗建议 | - | "建议先从内部转岗开始" |
| `blood_relationship` | VARCHAR(50) | varchar(50) | - | 血缘关系(岗位关联性) | - | "近亲", "远亲", "无关联" |
| `is_recommended` | BOOLEAN | boolean | DEFAULT false | 是否推荐 | ✅ | true, false |
| `created_at` | TIMESTAMPTZ | timestamp | NOT NULL, DEFAULT NOW | 创建时间 | - | 2024-01-15T10:30:00+08:00 |

**示例数据**:
```json
{
  "id": 1,
  "from_job_id": 1,
  "to_job_id": 10,
  "path_type": "related",
  "intermediate_job_id": null,
  "transfer_difficulty": "medium",
  "steps_required": 1,
  "additional_skills": "需要学习产品设计, 用户研究, 数据分析",
  "additional_certificates": "推荐考取NPDP或PMP认证",
  "transfer_conditions": "1. 有2年以上前端开发经验\n2. 参与过完整产品周期\n3. 对产品有浓厚兴趣",
  "salary_impact": "转岗初期可能薪资持平或略降(0-15%), 1-2年后可追平反超",
  "transfer_tips": "1. 先在公司内部寻找转岗机会\n2. 业余时间学习产品相关知识\n3. 尝试从产品助理开始过渡",
  "blood_relationship": "近亲",
  "is_recommended": true,
```

## PostgreSQL 数据类型对照

| PostgreSQL类型 | Drizzle类型 | TypeScript类型 | 说明 |
|--------------|------------|---------------|------|
| `SERIAL` | `serial` | `number` | 自增整数主键 |
| `INTEGER` | `integer` | `number` | 整数 |
| `VARCHAR(n)` | `varchar(n)` | `string` | 变长字符串 |
| `TEXT` | `text` | `string` | 长文本 |
| `BOOLEAN` | `boolean` | `boolean` | 布尔值 |
| `TIMESTAMPTZ` | `timestamp` | `string` | 带时区的时间戳 |

## 索引设计

### jobs 表索引
| 索引名 | 字段 | 用途 |
|--------|------|------|
| `jobs_job_title_idx` | `job_title` | 按岗位名称搜索 |
| `jobs_address_idx` | `address` | 按地点搜索 |
| `jobs_company_name_idx` | `company_name` | 按公司名称搜索 |
| `jobs_industry_idx` | `industry` | 按行业筛选 |
| `jobs_job_code_idx` | `job_code` | 按岗位代码查询 |

### job_profiles 表索引
| 索引名 | 字段 | 用途 |
|--------|------|------|
| `job_profiles_category_idx` | `category` | 按分类筛选 |
| `job_profiles_name_idx` | `profile_name` | 按画像名称搜索 |

### career_paths 表索引
| 索引名 | 字段 | 用途 |
|--------|------|------|
| `career_paths_from_job_idx` | `from_job_id` | 查询某个岗位的晋升路径 |
| `career_paths_to_job_idx` | `to_job_id` | 查询到某个岗位的路径 |

### job_transfer_graph 表索引
| 索引名 | 字段 | 用途 |
|--------|------|------|
| `job_transfer_graph_from_job_idx` | `from_job_id` | 查询某个岗位的转岗路径 |
| `job_transfer_graph_to_job_idx` | `to_job_id` | 查询到某个岗位的路径 |
| `job_transfer_graph_recommended_idx` | `is_recommended` | 筛选推荐的路径 |
