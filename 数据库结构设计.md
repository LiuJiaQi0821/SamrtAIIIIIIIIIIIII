# 数据库结构设计

## 数据库类型
- **PostgreSQL** (通过 Supabase 托管)
- **ORM**: Drizzle ORM

---

## 数据表概览

| 表名 | 说明 | 主要用途 |
|------|------|---------|
| `health_check` | 健康检查表 | 系统监控用 |
| `jobs` | 岗位数据表 | 存储招聘岗位信息 |
| `job_profiles` | 岗位画像表 | 存储岗位能力要求和画像 |
| `career_paths` | 岗位晋升路径表 | 存储岗位之间的晋升关系 |
| `job_transfer_graph` | 换岗路径图谱表 | 存储岗位之间的转岗关系 |

---

## 详细表结构

### 1. health_check (健康检查表)
**系统表，禁止修改**

| 字段名 | 类型 | 说明 |
|--------|------|------|
| `id` | serial | 主键 |
| `updated_at` | timestamp | 更新时间(带时区) |

---

### 2. jobs (岗位数据表)
**存储招聘岗位信息**

| 字段名 | 类型 | 约束 | 说明 | 索引 |
|--------|------|------|------|------|
| `id` | serial | PRIMARY KEY | 主键ID | - |
| `job_title` | varchar(255) | NOT NULL | 岗位名称 | ✅ |
| `address` | varchar(255) | NOT NULL | 工作地点/地址 | ✅ |
| `salary_range` | varchar(100) | - | 薪资范围 | - |
| `company_name` | varchar(255) | NOT NULL | 公司名称 | ✅ |
| `industry` | varchar(255) | - | 所属行业 | ✅ |
| `company_size` | varchar(100) | - | 公司规模 | - |
| `company_type` | varchar(100) | - | 公司类型 | - |
| `job_code` | varchar(100) | - | 岗位代码 | ✅ |
| `job_description` | text | - | 岗位描述 | - |
| `update_date` | varchar(50) | - | 更新日期 | - |
| `company_description` | text | - | 公司描述 | - |
| `source_url` | varchar(500) | - | 来源链接 | - |
| `created_at` | timestamp | NOT NULL, DEFAULT NOW | 创建时间 | - |

---

### 3. job_profiles (岗位画像表)
**存储岗位能力要求和画像**

| 字段名 | 类型 | 约束 | 说明 | 索引 |
|--------|------|------|------|------|
| `id` | serial | PRIMARY KEY | 主键ID | - |
| `profile_name` | varchar(100) | NOT NULL | 画像名称 | ✅ |
| `category` | varchar(50) | NOT NULL | 分类 | ✅ |
| `description` | text | - | 描述 | - |
| `professional_skills` | text | - | 专业技能要求 | - |
| `certificate_requirements` | text | - | 证书要求 | - |
| `innovation_ability` | integer | DEFAULT 3 | 创新能力(1-5分) | - |
| `learning_ability` | integer | DEFAULT 3 | 学习能力(1-5分) | - |
| `pressure_resistance` | integer | DEFAULT 3 | 抗压能力(1-5分) | - |
| `communication_ability` | integer | DEFAULT 3 | 沟通能力(1-5分) | - |
| `internship_requirement` | text | - | 实习经验要求 | - |
| `education_requirement` | varchar(100) | - | 学历要求 | - |
| `work_experience_requirement` | varchar(100) | - | 工作经验要求 | - |
| `salary_range` | varchar(100) | - | 薪资范围 | - |
| `career_direction` | text | - | 就业方向 | - |
| `core_qualifications` | text | - | 核心素质要求 | - |
| `tools_requirement` | text | - | 工具/软件要求 | - |
| `created_at` | timestamp | NOT NULL, DEFAULT NOW | 创建时间 | - |

---

### 4. career_paths (岗位晋升路径表)
**存储岗位之间的晋升关系**

| 字段名 | 类型 | 约束 | 说明 | 索引 |
|--------|------|------|------|------|
| `id` | serial | PRIMARY KEY | 主键ID | - |
| `from_job_id` | integer | NOT NULL, FK | 当前岗位ID (关联 job_profiles) | ✅ |
| `to_job_id` | integer | NOT NULL, FK | 目标岗位ID (关联 job_profiles) | ✅ |
| `path_type` | varchar(20) | NOT NULL | 路径类型: vertical(垂直晋升)/horizontal(横向转岗) | - |
| `difficulty` | varchar(20) | DEFAULT 'medium' | 难度: easy/medium/hard | - |
| `years_required` | integer | DEFAULT 3 | 所需周期(年) | - |
| `additional_skills` | text | - | 所需补充技能 | - |
| `additional_certificates` | text | - | 所需补充证书 | - |
| `promotion_conditions` | text | - | 晋升条件说明 | - |
| `salary_changes` | text | - | 薪资变化说明 | - |
| `path_description` | text | - | 路径描述 | - |
| `is_recommended` | boolean | DEFAULT false | 是否推荐 | - |
| `created_at` | timestamp | NOT NULL, DEFAULT NOW | 创建时间 | - |

---

### 5. job_transfer_graph (换岗路径图谱表)
**存储岗位之间的转岗关系**

| 字段名 | 类型 | 约束 | 说明 | 索引 |
|--------|------|------|------|------|
| `id` | serial | PRIMARY KEY | 主键ID | - |
| `from_job_id` | integer | NOT NULL, FK | 起始岗位ID (关联 job_profiles) | ✅ |
| `to_job_id` | integer | NOT NULL, FK | 目标岗位ID (关联 job_profiles) | ✅ |
| `path_type` | varchar(20) | NOT NULL | 路径类型 | - |
| `intermediate_job_id` | integer | FK | 中间岗位ID (关联 job_profiles) | - |
| `transfer_difficulty` | varchar(20) | DEFAULT 'medium' | 转岗难度: easy/medium/hard | - |
| `steps_required` | integer | DEFAULT 1 | 所需步骤数 | - |
| `additional_skills` | text | - | 所需补充技能 | - |
| `additional_certificates` | text | - | 所需补充证书 | - |
| `transfer_conditions` | text | - | 转岗条件说明 | - |
| `salary_impact` | text | - | 薪资影响 | - |
| `transfer_tips` | text | - | 转岗建议 | - |
| `blood_relationship` | varchar(50) | - | 血缘关系(岗位关联性) | - |
| `is_recommended` | boolean | DEFAULT false | 是否推荐 | ✅ |
| `created_at` | timestamp | NOT NULL, DEFAULT NOW | 创建时间 | - |

## 索引设计

### jobs 表索引
- `jobs_job_title_idx` - 岗位名称索引
- `jobs_address_idx` - 地点索引
- `jobs_company_name_idx` - 公司名称索引
- `jobs_industry_idx` - 行业索引
- `jobs_job_code_idx` - 岗位代码索引

### job_profiles 表索引
- `job_profiles_category_idx` - 分类索引
- `job_profiles_name_idx` - 画像名称索引

### career_paths 表索引
- `career_paths_from_job_idx` - 起始岗位索引
- `career_paths_to_job_idx` - 目标岗位索引

### job_transfer_graph 表索引
- `job_transfer_graph_from_job_idx` - 起始岗位索引
- `job_transfer_graph_to_job_idx` - 目标岗位索引
- `job_transfer_graph_recommended_idx` - 推荐标志索引

## 技术栈
- **数据库**: PostgreSQL (Supabase)
- **ORM**: Drizzle ORM
- **连接方式**: Supabase JavaScript Client
- **环境变量**:
  - `COZE_SUPABASE_URL` - Supabase URL
  - `COZE_SUPABASE_ANON_KEY` - 公开密钥
  - `COZE_SUPABASE_SERVICE_ROLE_KEY` - 服务端密钥(可选)
